# architected-example-observable

> Observable example for architected

## Usage

```js
npm link
```

Link example.

```js
architected-example-observable
```

Run example! ⚠️ It will *not* override anything.

## Cli

```bash
# specify path
architected-example-observable --path ./test
```

## Uninstall

```bash
npm r -g architected-example-observable
```
