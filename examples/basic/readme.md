# architected-example-basic

> Basic example for architected

## Usage

```js
npm link
```

Link example.

```js
architected-example-basic
```

Run example! ⚠️ It will *not* override anything.

## Cli

```bash
# specify path
architected-example-basic --path ./test
```

## Uninstall

```bash
npm r -g architected-example-basic
```
